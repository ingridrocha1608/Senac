# Senac
Repositorio dedicado aos exercícios do curso de tecnologia em informática para internet

* Repositórios de Aulas – Tecnologia de Informática para a Internet

Este repositório reúne os materiais produzidos durante o curso de Tecnologia de Informática para a Internet.
O objetivo é organizar e registrar todo o conteúdo estudado em aula por meio de exercícios práticos e pequenos projetos.

* O que você encontrará aqui

Atividades de HTML, CSS e JavaScript
Exercícios de estruturação de páginas, estilização e criação de interações básicas.

Projetos práticos desenvolvidos em sala
Como calculadoras, páginas interativas, formulários e miniaplicações.

Arquivos de estudo e experimentação
Testes de código e exemplos criados para fixação dos conteúdos.

* Finalidade

Manter um histórico claro da evolução no curso, facilitar revisões futuras e permitir que qualquer pessoa visualize como os conceitos foram aplicados na prática.
